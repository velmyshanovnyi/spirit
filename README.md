# Spirit

Децентралізований, анонімний P2P-месенджер без постійної серверної інфраструктури, баз даних та ідентифікаційних даних користувача.

## Філософія

- **Анонімність пристрою** — акаунт це виключно локальна пара криптографічних ключів у пам'яті браузера. Без email, телефону, OAuth.
- **Ефемерність та мультиакаунтність** — постійні зашифровані профілі або одноразові "світи", що зникають безслідно після закриття вкладки.
- **Zero-database бекенд** — сигналінг координує встановлення з'єднання через легкий PHP-скрипт із файловим сховищем, що самоочищається. Ніякі повідомлення через сервер не проходять.
- **Селективна інфраструктура** — користувач сам обирає сигнальний та STUN/TURN сервер.
- **E2EE за замовчуванням** — весь трафік у WebRTC DataChannel додатково шифрується end-to-end, сигнальний сервер бачить лише публічні ключі та SDP/ICE, ніколи вміст переписки.
