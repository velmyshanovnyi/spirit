# Спека: Основа редизайну — ідентикони, іконки довіри, gear-іконка (UI)

Перший крок уточненої концепції редизайну (палітра "Кур'єр" + фігурні аватари "Тінь" + іконки довіри замість тексту), узгодженої з користувачем через макет (`https://claude.ai/code/artifact/384f08c4-450f-4aef-9645-7a6d70e55205`). Навмисно НЕ включає велику структурну перебудову навігації (постійний Telegram-подібний сайдбар) чи дерево папок — це окремі, більші наступні кроки. Цей крок — найменш ризикована, додаткова візуальна основа над наявним екраном контактів, без зміни моделі навігації.

## Дизайн-рішення (перенесено з макета, узгоджено з користувачем)

1. **Ідентикони**: детермінований візерунок з fingerprint-хешу контакту (5×3 сітка, дзеркальна по горизонталі), той самий алгоритм, що в макеті. Форма контейнера кодує тип: коло — звичайний контакт (identity-announce verified), квадрат із заокругленими кутами — група (`groups.js`), хвилястий контур (легка "дихаюча" CSS-анімація з `prefers-reduced-motion`-запобіжником) — ефемерна личность (сесія без vault key).
2. **Іконки довіри замість тексту**: щит-іконка з 2 реальними станами на екрані контактів (третій стан макета — "imported" — стосується позначок ПОВІДОМЛЕНЬ, вже реалізовано в Секції I3 як `imported-badge`, не рядка контакту): заповнений щит з "✓" — контакт має хоча б один верифікований proof (`contact.proofSet.proofs.length > 0` і хоч один `verifiedAt`); контурний щит без "✓" — TOFU-контакт без жодного верифікованого proof (замінює наявний `contacts.unverified` текстовий бейдж).
3. **Gear-іконка налаштувань**: inline SVG (Telegram-подібна шестерня) замість емодзі "⚙️" на кнопці `#btn-settings-toggle`.

## Верифікація

RF1 (чиста функція) — безпечно почати без додаткових питань. RF2 — жива перевірка на обох хостах: аватари й щити рендеряться коректно, налаштування відкриваються при кліку на нову gear-іконку (регресія наявного меню), консольних помилок немає.

## Наступні кроки (поза цією секцією, для довідки)

Дерево папок з drag&drop, постійний Telegram-подібний сайдбар (заміна моделі "один екран за раз"), фільтри Усі/Верифіковані/Групи, кільце прогресу на групових аватарах, панель налаштувань як список — заплановані окремі майбутні секції, не в цій спеці.
