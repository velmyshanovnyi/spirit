## Знахідки

Немає нових. Усі 6 фіксів ітерації 1 підтверджено коректними, зокрема:
- Обхід nonce/audience **повністю закрито**: жодного шляху для випадкового збігу немає, бо `payload.nonce`/`payload.aud` автентифіковані підписом Google — атакуючий не може підробити ці значення, а порожній/whitespace `expectedNonce` з боку викликача — це баг викликача, не обхід атакуючого.
- 60с clock-skew допуск стандартний і безпечний для login-атестації; симетрична перевірка `iat` не створює ризику хибної відмови для реальних Google-токенів.
- Порядок перевірок підтверджено: пошкоджений base64url/не-об'єктний JSON падає всередині `decodeJwt` **до** будь-якого мережевого/крипто виклику.
- Документаційний guard на `jwks`-параметр визнано достатнім для цієї ітерації — реального виклику з UI (Секція 2) ще немає в коді.
